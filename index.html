安装Windows 11需要注意什么？

发表于 2022-11-18 更新于 2022-12-17

硬件要求

  • 确保设备有TPM 2.0模块
  • CPU在支持列表内,详情见Windows 处理器要求 | Microsoft Learn
  • 22H2版本Windows 11全新安装时必须连接网络

绕过限制

  • 如果你的设备硬件达不到要求，可以使用Rufus制作可绕过限制的镜像
  • 如果你不想在22H2版本Windows 11的OOBE步骤时登录Microsoft 帐户，可以在帐户处输入“1@1.com”，密码处随意输入，即可跳过。

删除Windows上保留的存储

发表于 2022-11-17 更新于 2022-12-17

较新版本的Windows 10（大概在v1903以后）和Windows 11上有占用极大的“保留的存储”，可以用Dism来管理
打开CMD或PowerShell
输入“DISM.exe /Online /Get-ReservedStorageState”查询是否已启用
输入“DISM.exe /Online /Set-ReservedStorageState /State:Disabled”禁用
输入“DISM.exe /Online /Set-ReservedStorageState /State:Enabled”启用
